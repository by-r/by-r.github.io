## How to run locally

### Docker (Recommended) 

- run 
```bash
  docker compose pull
  docker compose up
```
- head over to `http://localhost:8000` or the address shown in your console

### Others

- Assuming you have Ruby and Bundler installed.

```bash
  bundle install
  pip install jupyter
  bundle exec jekyll serve --lsi

```
